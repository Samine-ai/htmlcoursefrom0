Que és HTML?

HTML significa Hyper Text Markup Language. Es el lenguaje básico para crear páginas web. Con HTML, le damos estructura y contenido a una página. Tal i como yo mismo lo estoy haciendo con esta pagina web, que solo estara hecha con HTML

Basicamente podriamos decir que HTML es el esqueleto de la pagina web, por lo que no podremos hacer animaciones ni que quede bonito ya que eso se hace con otras paginas web como CSS.

Como funciona?

HTML digamos que usa etiquetas, son como cajas donde dentro ponemos lo que queremos que contenga.
Hay 2 tipos de etiquetas unas que se abren i se cierran donde dentro podemos añadir o cambiar el contenido.
Cuando se crea una página web, se escribe en un archivo con la extensión .html, y al abrirlo en un navegador (como Chrome, Firefox o Edge), el navegador lee el archivo línea por línea, interpreta las etiquetas y genera la visualización de esa página en la pantalla.

Que tienes que saber?

Primero, quiero que entiendas qué es lo que vamos a aprender.
HTML no es complicado. Es como aprender a escribir instrucciones para que el navegador (como Chrome o Firefox) muestre texto, imágenes, botones, enlaces, formularios y cualquier otro contenido en una página web.
Lo que vas a hacer con HTML es decirle al navegador: “esto es un título”, “esto es un párrafo”, “esto es una imagen”, y así.
Tú no estás programando, estás estructurando contenido. Como poner etiquetas en cajas para organizarlas.
Nada de matemáticas, nada de fórmulas, todo son cajas i etiquetas donde poner lo que tu quieras.
